== Installation ==
= FROM WITHIN WORDPRESS =
1. Visit 'Plugins > Add New'.
2. Search for ‘Venby’.
3. Activate ‘Venby’ from your Plugins page.
4. Go to "after activation" below.

=MANUALLY=
1. Upload the Venby Plugin folder to the /wp-content/plugins/directory
2. Activate the ‘Venby’ plugin through the ‘Plugins’ menu in WordPress
3. Go to “after activation” below.

=AFTER ACTIVATION=
1. You should see a new link called ‘Venby’ in your Wordpress settings.
2. Read the instructions and set up your account with your Venby API key (found in the Venby app admin)
3. Embed or share the link in your campaign in any of your Wordpress pages or posts

== Frequently Asked Questions ==

=Is Stripe required?=

Yes, Venby is built on Stripe so products, customers, and orders are imported & exported using Stripe’s API. You can create new products using Stripe or import new products with Google Shopping, Shopify, or Woocommerce.

=Is Venby free?=

Every account includes 1 free campaign. You can create unlimited campaigns with a paid plan.

=Is Venby secure?=

Yes, the Venby eCommerce pop ups are loaded in a secure HTTPS iframe. We use Stripe’s latest API to handle all credit card transactions. Stripe is the leading credit card payment processors on the web. Stripe handles all sensitive financial information. Venby does not save or store any credit cards. If it’s good enough for Facebook & Pinterest, it’s good enough for us.

=What is a campaign?=

A campaign is 1 image or video with an associated product. A single video can link multiple products. However, a single image is linked with a single product.

=Do I upload my videos or images directly to Venby?=

Yes. We host your video and image for each campaign.

=Can I run my Venby campaign on any website?=

Yes. Your campaigns can run across multiple websites. You can control which sites display each of your campaigns using our whitelist permissions.

=Can I embed campaigns on social networks like Facebook or Instagram?=

Your campaigns cannot run within social sites like Facebook. However, You can share a link to any dedicated landing page or your own website.

=What credit cards does Venby accept?=

We use Stripe for payment processing. Stripe can accept all major debit and credit cards from customers in every country in 135+ currencies.  You can also use 3D Secure for any payment.

=Does Venby take a sale percentage?=

No, Venby does not take a percentage of each sale. Depending on volume, Stripe usually takes a 2.9% cut per transaction.
